# 👑 Chess Analyzer & AI Coach (Chess.com Dark Clone)

Интерактивный шахматный анализатор партий с полноценным ИИ-тренером на базе **Gemini 3.1 Flash Lite** и движка **Stockfish**. Интерфейс полностью кастомизирован под премиальный темный дизайн Chess.com.

---

## 🚀 Основные фичи

*   **🧠 Интерактивный тренер Gemini:** ИИ не просто оценивает позицию, а объясняет ошибки человеческим языком. Чтобы исключить "шахматную слепоту" нейросетей, бэкенд передает в Gemini ход-опровержение от Stockfish — тренер бьет точно в цель!
*   **💎 Алгоритм Бриллиантовых ходов (`!!`):** Прямо как на Chess.com! Система анализирует твои ходы, и если ты делаешь тактически оправданную жертву материала, которая ведет к победе, интерфейс взрывается бирюзовым неоновым свечением.
*   **🏖️ Умная «Песочница» и Боковые ветки (Sidelines):** Ты можешь в любой момент отклониться от главной PGN-партии и начать двигать фигуры руками. Система автоматически создаст временную ветку, перестроит график оценок, а кнопка возврата вернет тебя на главную линию.
*   **📊 Тотальный Game Review (Отчет о партии):** 
    *   Разделение панели на удобные вкладки `[ Ходы ]` и `[ Отчет ]` (всё на одном экране без скролла).
    *   Честная формула точности (Accuracy %) с защитой от "взрыва" CPL при матах.
    *   График Chart.js в реальном времени.
    *   Динамические эмоции тренера (😱 при зевках, 🔥 при бриллиантах).
    *   Определитель шахматных дебютов.

---

## 🛠️ Технологический стек

*   **Backend:** Python, FastAPI, Uvicorn, Stockfish Engine.
*   **Frontend:** HTML5, CSS3 (Neon Dark Theme), JavaScript.
*   **Библиотеки:** `chessboard.js` (интерфейс доски), `chess.js` (валидация правил ФИДЕ), `Chart.js` (плавный график оценки).

## 📦 Как запустить проект локально

1. Склонируй репозиторий:
```bash
git clone <ссылка_на_твой_гит>
cd <имя_папки>
```

2. Установи зависимости (Python 3.10+):
```bash
pip install -r requirements.txt
```

3. Убедись, что у тебя установлен и прописан в путях движок Stockfish.

4. Добавь свой API-ключ Gemini в переменные окружения.

5. Запусти сервер:
```bash
uvicorn main:app --reload
```

6. Открой в браузере http://127.0.0.1:8000 и наслаждайся анализом!
